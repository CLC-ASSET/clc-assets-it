# نظام إدارة أصول تقنية المعلومات - CLC

هذه نسخة مبدئية Front-End باللغة العربية لإدارة:
- أجهزة Laptop / Desktop
- الشاشات
- الطابعات
- الصيانة
- التسليم والاستلام
- البحث والفلاتر
- صلاحية مستخدم عادي بدون حذف

## طريقة التشغيل
افتح ملف index.html مباشرة من المتصفح.

## مهم
هذه نسخة تجريبية تعمل على المتصفح وتخزن البيانات محلياً في LocalStorage.
للنسخة الأونلاين الحقيقية يجب ربطها بـ:
- Google Sheets + Apps Script
أو
- Microsoft Lists / SharePoint
أو
- قاعدة بيانات مثل Firebase / Supabase

## الصلاحيات
- المستخدم العادي: إضافة فقط.
- المدير: تعديل وحذف وتصدير.
